Prompt Sentinel es un sistema de proteccion multi-capa disenado para detectar y bloquear ataques de inyeccion de instrucciones en modelos de lenguaje (LLMs). Combina filtros heuristicos, un modelo de Deep Learning y un LLM-Judge para ofrecer una defensa robusta y escalable.

Arquitectura del Sistema

Cada prompt es analizado secuencialmente por 3 capas independientes que proporcionan defensa en profundidad.

01

Capa 1: Filtro Heuristico

Analisis rapido basado en expresiones regulares, deteccion de caracteres invisibles, homoglifos, bases64 y scoring de perplejidad. Filtra el 90% de prompts obviamente maliciosos en milisegundos.

Regex Perplejidad Homoglifos Zero-width
02

Capa 2: Modelo DistilBERT

Clasificador binario fine-tuneado con miles de ejemplos de prompts benignos y maliciosos. Detecta patrones sutiles de inyeccion que el filtro heuristico podria pasar por alto.

Deep Learning Fine-tuned Binario
03

Capa 3: LLM-Judge (Mistral)

Analisis semantico avanzado utilizando Mistral como juez. Evalua la intencion, contexto y posible impacto del prompt, proporcionando la decision final con explicaciones detalladas.

Caracteristicas Principales

Disenado para ser rapido, preciso y facil de integrar en cualquier sistema que utilice LLMs.

Deteccion en Tiempo Real

Analiza y clasifica prompts en menos de 3 segundos, permitiendo proteger aplicaciones en tiempo real sin degradar la experiencia del usuario.

Defensa en Profundidad

Tres capas independientes se complementan entre si, minimizando tanto falsos positivos como falsos negativos en la deteccion.

Dashboard de Analisis

Visualiza en tiempo real los patrones de ataque, fuentes sospechosas y metricas de rendimiento del sistema de deteccion.

API RESTful

Integracion sencilla via API REST con documentacion interactiva (Swagger/OpenAPI). Compatible con cualquier stack tecnologico.

Registro de Ataques

Cada intento de inyeccion queda registrado en una base de datos SQL para analisis forense, metricas y mejora continua del sistema.

Despliegue Containerizado

Listo para produccion con Docker y Docker Compose. Despliegue en un solo comando en cualquier infraestructura.
