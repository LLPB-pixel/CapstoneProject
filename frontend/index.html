Sistema de Seguridad

Prompt Guard es un sistema de protección multi-capa diseñado para detectar y bloquear ataques de inyección de instrucciones en modelos de lenguaje (LLMs). Combina filtros heuristicos, un modelo de Deep Learning y un LLM-Judge para ofrecer una defensa robusta y escalable.

Arquitectura del Sistema

Cada prompt es analizado secuencialmente por 3 capas independientes que proporcionan defensa en profundidad.

01

Capa 1: Filtro Heurístico

Análisis rápido basado en expresiones regulares, detección de caracteres invisibles, homoglifos, bases64 y scoring de perplejidad. Filtra el 90% de prompts obviamente maliciosos en milisegundos.

Regex Perplejidad Homoglifos Zero-width
02

Capa 2: Modelo DistilBERT

Clasificador binario fine-tuneado con miles de ejemplos de prompts benignos y maliciosos. Detecta patrones sutiles de inyección que el filtro heurístico podría pasar por alto.

Deep Learning Fine-tuned Binario
03

Capa 3: LLM-Judge

Análisis semántico avanzado mediante un LLM especializado como juez. Evalúa la intención, contexto y posible impacto del prompt, proporcionando la decision final con explicaciones detalladas.

Características Principales

Diseñado para ser rápido, preciso y fácil de integrar en cualquier sistema que utilice LLMs.

Detección en Tiempo Real

Analiza y clasifica prompts en menos de 3 segundos, permitiendo proteger aplicaciones en tiempo real sin degradar la experiencia del usuario.

Defensa en Profundidad

Tres capas independientes se complementan entre si, minimizando tanto falsos positivos como falsos negativos en la detección.

Dashboard de Análisis

Visualiza en tiempo real los patrones de ataque, fuentes sospechosas y métricas de rendimiento del sistema de detección.

API RESTful

Integración sencilla vía API REST con documentación interactiva (Swagger/OpenAPI). Compatible con cualquier stack tecnologico.

Registro de Ataques

Cada intento de inyección queda registrado en una base de datos SQL para análisis forense, métricas y mejora continua del sistema.

Despliegue Containerizado

Listo para producción con Docker y Docker Compose. Despliegue en un solo comando en cualquier infraestructura.
